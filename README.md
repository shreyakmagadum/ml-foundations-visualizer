# 📊 ML Foundations Visualizer

## 📌 Project Description
This project is an interactive Streamlit app to visualize core ML concepts like vectors, derivatives, and gradient descent.

## 🚀 Features
- Vector similarity (dot product, cosine, distance, angle)
- Function and derivative visualization
- Gradient descent simulation
- Interactive UI

## 🛠 Tech Stack
- Python
- NumPy
- SymPy
- Plotly
- Streamlit

## ▶️ How to Run
pip install -r requirements.txt  
streamlit run app.py
